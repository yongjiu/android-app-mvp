# android-app-mvp
Yet Another MVP Framework for Android.

```
android.app.mvp.example                 (package)
|---model                               (package)
|   |---impl                            (package)
|   |   |---XxxxModelImpl.java          (class implements XxxxModel)
|   |   |---...                         (class)
|   |---XxxxModel.java                  (interface extends IModel)
|   |---...                             (interface)
|---view                                (package)
|   |---activity                        (package)
|   |   |---XxxxActivity.java           (class implements XxxxView)
|   |   |---...                         (class)
|   |---fragment                        (package)
|   |   |---XxxxFragment.java           (class implements XxxxView)
|   |   |---...                         (class)
|   |---adapter                         (package)
|   |   |---...                         (class)
|   |---widget                          (package)
|   |   |---...                         (class)
|   |---XxxxView.java                   (interface extends IView)
|   |---...                             (interface)
|---presenter                           (package)
    |---XxxxPresenter.java              (class implements IPresenter)
    |---...                             (class)
```
